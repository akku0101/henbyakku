This version was created by replacing some files contained within the dev-flash directory of firmware 4.82. 

Some users have already installed this version, although it would be completely useless without a new HEN update. 

In the meantime, it will take some time to update HEN, probably more for the bgtoolset suite of tools which will need some patches contained in a future Custom Firmware 4.89.


On Twitter, Real Yoti released the official modified firmware HFW 4.89.2, which is based on the PS3's 4.89 and can be installed on all PS3 models.

HFW 4.89.1 released by Louay was once withdrawn due to a problem, but this time Real Yoti (Yoti) built and distributed HFW. The version is HFW 4.89.2, which means it is the successor to Louay's version.


The content that Yoti posted on the PSX-Place forum is the information of the difference file between the original 4.82 HFW and 4.89 HFW.

The differences between 4.82 and 4.89 are as follows.

autodownload_plugin.sprx
avc_plugin.sprx
gameupdate_plugin.sprx
nas_plugin.sprx
newstore_plugin.sprx
silk_webkit.sprx
This time it seems that it is HFW that can be installed without problems.
